Twizo Verification is a 2FA plugin made for WordPress and WooCommerce.

== Description ==
Twizo Verification is a 2FA plugin made for WordPress and WooCommerce, it adds a new layer of security to your website. With this plugin you can easily add 2FA to your WordPress site. When you are using the WooCommerce plugin, you can also enable 2FA for your customers.

The WooCommerce plugin is not a requirement for the Twizo Verification plugin, it is optional!

When installed, user can do the following:

* Activate 2FA in the WordPress and WooCommerce My Account page.
* Login with the chosen or default authentication method. (Twizo Authenticator App, Bio Voice, SMS, Voice call, Messaging clients (Facebook Messenger, Telegram, LINE) or Backup Codes).
* Change their phone number.
* Generate and use backup codes to login when their authentication device is lost.

As an admin you can:
* Set the default authentication method.
* Change the logo image to make the authentication widget suit your style more.
* Set a sender which your customers will see when receiving a SMS.
* Select the API host to use.
* Set the API key.
* The plugin on the My Account page uses the standard WooCommerce styling, so you can style it how you want.

== Installation ==
* Download the plugin from the Wordpress Plugin Directory.
* Activate the plugin under \'Plugins\' in Wordpress.
* Go to the plugins page in the side bar.
* The Plugin will walk you through the steps of setting up Twizo Verification for WordPress and WooCommerce.
